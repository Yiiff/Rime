# Rime
> 此Repo为自家用鼠须管配置文件，说白了就是懒，还了台电脑之后才想起来要备份。

使用方法：

1. `git clone https://github.com/ZX-ZhouXiao/Rime.git `
2. `cp -f Rime/* /Users/Your Account Name/Library/Rime`


**注意：Your Account Name为你电脑磁盘根目录的名称。**

查看方式：

1. Open Finder
2. 键入` CMD + ,`
3. 找到下图箭头所指的内容即可
4. 重新部署鼠须管

![Screenshot](https://github.com/ZX-ZhouXiao/MarkDown-Photos/blob/master/RimeScreenShot.png?raw=true)

And enjoy!
